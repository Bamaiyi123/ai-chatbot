# 🤖 AI-Powered Smart Chatbot with Voice & Document Intelligence

## 🚀 Overview

This project is an advanced AI chatbot application that combines natural
language processing, document intelligence, and voice interaction into a
single platform.

The chatbot is capable of holding context-aware conversations, analyzing
PDF documents, performing semantic search, and interacting through voice
input and output.

## ✨ Features

-   💬 Conversational AI (with memory)
-   📄 PDF Document Intelligence
-   🧠 Semantic Search (FAISS)
-   🎤 Voice Interaction (audio upload + playback)
-   🌐 Live Deployment

## 🛠️ Tech Stack

-   Frontend/UI: Streamlit
-   Backend: Python
-   LLM API: Groq
-   Embeddings: Sentence Transformers
-   Vector DB: FAISS
-   PDF Processing: PyPDF
-   Voice Output: gTTS

## 📂 Project Structure

advanced-chatbot/ │ ├── app.py ├── requirements.txt ├── README.md └──
.env (not included)

## ⚙️ Setup

1.  Clone repo
2.  Create virtual environment
3.  Install dependencies: pip install -r requirements.txt
4.  Add API key in .env: OPENAI_API_KEY=your_groq_api_key
5.  Run: streamlit run app.py

## 🌐 Deployment

Deploy using Streamlit Cloud and connect GitHub repo.
